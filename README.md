# Mernaapp
Application build completely using mern stack .

# Deployed using netlify and heroku
Link - https://thefoodquest.netlify.app/

The technology used on client side are:
REACT
REACT-ROUTER-DOM
REDUX
FETCH 

The technology used on server side are:
NODEJS
EXPRESS
MONGODB ATLAS
